# ERP-Lite 核心操作指南

本文档旨在帮助您快速上手并全面了解 ERP-Lite 租赁管理系统的各项功能。系统通过 **"销售商品"** 与 **"库存资产"** 分离的核心设计，通过 **BOM (物料清单)** 进行智能关联，实现灵活的租赁套餐组合与精准的库存调度。

---

## 1. 核心概念：使用前必读

在开始操作前，请务必理解以下几个核心概念的区别与联系：

*   **商品 (Products)**：
    *   **定义**：对外展示给客户看的"上架销售的规格"。
    *   **包含**：商品名称、租赁价格规则、保险费、押金等。
    *   **示例**："索尼A7M4 婚礼拍摄套餐"、"iPhone 13 租赁"。

*   **资产 (Inventory Items)**：
    *   **定义**：仓库里实实在在的"库存实物"。
    *   **包含**：库存总量、序列号 (SN码)、当前资产状态（可用、租赁中等）。
    *   **示例**："索尼A7M4机身" (SN: 123456)、"原装电池" (不追踪SN)。

*   **BOM (物料清单)**：
    *   **定义**：连接"商品"与"资产"的桥梁。
    *   **作用**：告诉系统"租出去1个商品套餐，需要从仓库扣减哪些具体的库存资产"。
    *   **示例**：1个"索尼A7M4套餐" = 1个"A7M4机身" + 1个"24-70镜头" + 2块"电池"。

## 2. 快速入门流程
建议先完成库存与商品配置，再开始录单或接入线上抓取。

### 第一步：仓库与库存物品定义
1. 进入 **[商品管理] > [库存数量管理]**。
2. 如尚未创建仓库：在页面内的仓库管理区域先新增仓库（如“主仓”“分仓A”）。
3. 点击 **[新增物品定义]**，创建库存物品类型（如“iPhone 13 128G 机身”“A7M4 电池”）。
4. 关键字段说明：
   - **是否序列化**  
     - 序列化：一物一码，必须录入 SN（适用于手机/相机/机身等）
     - 非序列化：只管理数量（适用于耗材/配件/线材等）
   - **单位/分类/采购价**：用于展示与统计，建议按实际业务规范填写

### 第二步：入库（序列化与非序列化）
1. 进入 **[商品管理] > [库存数量管理]**。
2. **非序列化物品（数量管理）**：使用 **库存数量调整**，选择仓库与“入库/增加”“出库/减少”，输入数量并确认。
3. **序列化物品（SN 管理）**：
   - 使用 **快速入库** 或 **批量入库**：选择物品类型与仓库，按“每行一个 SN”录入。
   - 如需批量初始化：可使用 **导入序列物品 CSV**（格式见页面提示）。

### 第三步：创建商品规格与 BOM（决定库存日历准确性）
1. 进入 **[商品管理] > [商品规格管理]**。
2. 点击 **[添加规格]**，创建商品型号及其规格（对外展示/报价维度）。
3. 配置字段要点：
   - **保险费**、**价格规则（按天数）**：决定订单金额计算
   - **映射关键词**：用于线上订单导入/抓取后的规格自动匹配（建议把平台标题常见别名加入）
4. 在规格编辑中配置 **规格资产（BOM）**：
   - 为每个规格添加需要扣减的库存物品类型与数量
   - 未配置 BOM 的规格仍可开单，但库存日历会提示“无BOM”，且无法准确推算可接单量

### 第四步：线下订单录入与发货
1. 进入 **[订单管理] > [线下订单管理]**。
2. 点击 **[新建订单]**，填写客户信息、起租日/归还日、商品规格、金额等并保存。
3. 按订单状态推进业务流转：
   - **待审核**：审核通过/驳回
   - **待发货**：录入收件信息与物流信息，发货并分配 SN（序列化库存）
   - **待归还/归还中**：归还登记，必要时补录逾期费用/续租
   - **已完成/已关闭**：完成或关闭订单

### 第五步（可选）：接入线上订单抓取/导入
1. 在 **[系统管理] > [系统设置]** 生成并复制 **API Token**。
2. 在桌面工具 Sync Tool 的 **连接配置**中填写 ERP 地址与 Token 并保存连接。
3. 在 **[订单管理] > [线上订单管理]** 配置各平台站点与选择器，或点击 **获取初始配置**快速填入默认值。
4. 通过 Sync Tool 或页面内的导入功能把线上订单写入系统，并在需要时完成规格匹配与 SN 分配。

## 3. 菜单与功能详解（按左侧菜单逐项说明）
本章以“菜单 -> 子功能 -> 操作步骤/输入输出/注意事项”的方式，覆盖 Web 端全部菜单与页面内功能。若账号权限不足，菜单可能不可见或按钮不可点。

### 3.0 导航与登录
#### 3.0.1 登录（/login）
1. 输入用户名与密码，点击 **[登录]**。
2. 登录成功后进入首页；登录失败会提示错误原因（账号/密码不正确等）。

#### 3.0.2 左侧菜单与操作指南入口
1. 左侧菜单支持折叠/展开；有子菜单的分组可展开显示子页面。
2. 左侧菜单底部提供 **操作指南**入口（用于查看系统内置/维护的帮助内容）。
3. 左侧底部显示当前账号信息；点击 **[退出登录]** 退出系统。

### 3.1 首页（/）
**用途**：快速查看订单概览与经营数据。

1. **近 7 天订单数/销售额**：统计近 7 天创建的订单数量与金额。
2. **累计订单数/销售额**：统计历史总计。
3. **订单状态分布**：按状态（待审核/待发货/待收货/待归还/逾期/归还中/已完成/已关闭等）聚合数量。
4. **客户来源统计**：按平台（小红书/闲鱼/抖音/线下/其他）统计订单数与营收。
5. **跳转入口**：点击“查看完整订单列表”进入线下订单管理。

### 3.2 订单管理

#### 3.2.1 线下订单管理（/orders）
**页面组成**：顶部统计与操作区 + 订单列表（含筛选、排序、分页）+ 订单详情/操作弹窗。

**A. 顶部操作**
1. **导出订单**：点击右上角 **[导出订单]**，选择“订单状态/结款状态”等条件，生成并下载 Excel。
2. **新建订单**：点击右上角 **[新建订单]**，填写客户信息、起租/归还、商品规格与金额并保存。

**B. 新建订单字段要点（常见业务口径）**
1. **商品/规格**：决定发货资产与库存占用；建议优先选“已配置规格资产（BOM）”的规格。
2. **起租日/归还日**：影响库存日历占用区间；如后续续租，会在原订单基础上追加租期。
3. **客户来源/平台**：用于统计（如小红书/闲鱼/抖音/线下/其他）。
4. **推广渠道/推广员**：用于结算统计；若选择“零售”等无需推广员的来源，推广员可不填。

**C. 订单列表筛选与定位**
1. 常用筛选：订单号、客户、推广员、商品/规格、SN、平台、状态、结款状态、日期区间等。
2. **匹配规格**：可筛选“已匹配规格/未匹配规格”，用于排查库存日历无法准确计算的订单。
3. 列表支持分页；部分字段支持悬停查看完整内容（如地址/备注/物流信息）。

**D. 状态流转与业务动作（按按钮文案）**
1. **通过/拒绝审核**：对“待审核”订单执行通过或拒绝（拒绝后通常会进入关闭状态，无法恢复）。
2. **发货**：对“待发货”订单点击 **[发货]**，填写：
   - 物流方式：顺丰速运 / 线下自提 / 其他  
   - 物流单号（选填）  
   - 设备 SN 码（选填；若需要精确到具体设备，建议填写或在库存侧分配）
3. **确认发货**：对“已发货待确认”订单点击 **[确认发货]**，用于补齐流程节点。
4. **归还**：对“待归还/已逾期”订单点击 **[归还]**，填写归还物流方式与归还单号（线下自提可不填单号）。
5. **关闭订单**：对未完成订单点击 **[关闭订单]**，需填写“关闭备注（必填）”。
6. **删除**：点击 **[删除]** 永久删除订单（不可撤销，谨慎使用）。

**E. 金额、结款与资料维护**
1. **结款**：列表中可将订单标记为“已结款/未结款”，用于后续结算统计口径。
2. **快照**：点击 **[快照]** 查看该订单关键字段快照（收货信息、设备信息等），便于对账核对。
3. **匹配规格（库存关联）**：当订单规格未匹配时，可在订单内执行规格匹配（选择商品与规格），用于库存日历与资产占用计算。

**F. 导出**
1. 使用顶部 **[导出订单]**，按弹窗条件导出 Excel（常用于财务对账或运营统计）。

#### 3.2.2 线上订单管理（/online-orders）
**用途**：导入/管理线上订单数据，并配置各平台抓取同步。

**A. 同步状态与手动控制**
1. 页面支持对单个平台执行 **开始同步** / **停止同步**。
2. 同步执行中会显示状态与加载提示；可查看同步日志（如页面提供“日志/详情”入口）。
3. 遇到平台风控/验证码卡死时，可点击 **[重启浏览器]** 重新启动该平台的抓取浏览器实例。

**B. 同步配置（侧滑面板：同步配置）**
同步配置分为三类 Tab：
1. **全局配置**：自动同步开关、间隔/并发、夜间模式、Webhook 地址等。
2. **站点配置**：选择站点后配置登录 URL、账号密码、最大翻页数等。
3. **选择器配置**：配置抓取所需的 DOM 选择器；支持直接编辑“选择器 JSON”。

**C. 获取初始配置（覆盖当前站点配置）**
1. 在“站点配置”中选择站点后，点击 **[获取初始配置]**。
2. 该操作需要二次确认：会覆盖当前站点的登录 URL 与全部选择器。
3. 应用后仍需点击底部保存使其生效。

**D. 订单导入/导出**
1. 支持 **导出 CSV**。
2. 支持 **导入 CSV**，导入过程会提示成功/失败条数。

**E. 数据维护（危险/高频操作）**
1. **清空订单**：按平台清空已导入的线上订单（不可撤销，通常仅管理员可用）。
2. **清空匹配**：清除规格匹配结果，便于重新匹配。
3. **导出 SN 模板**：用于线下补录/批量分配 SN 的辅助表格。

#### 3.2.3 远程验证（/online-orders/remote-auth）
**用途**：当平台出现扫码/滑块/风控验证导致抓取中断时，管理员可在网页端远程接管浏览器完成验证。

1. 选择平台站点（赞晨/诚赁/奥租/优品租/零零享/人人租）。
2. 点击 **触发同步**，等待浏览器启动并刷新截图。
3. 在截图区域可进行鼠标点击、拖拽（用于滑块），并可输入文本（如验证码/账号密码等）。
4. 建议保持“自动刷新”开启；交互过程中系统会自动节流请求，避免误操作。

### 3.3 推广管理

#### 3.3.1 推广人员（/promoters）
**用途**：维护推广员（渠道人员）信息，并用于订单归因与结算统计。

**A. 列表与筛选**
1. **筛选创建人**：用于多账号分工；管理员可按创建人过滤推广员。
2. **筛选渠道**：按“渠道类型/渠道配置”过滤（与“推广渠道”中维护的渠道配置一致）。
3. **分页**：支持切换每页行数与分页浏览。

**B. 新增/编辑推广员**
1. 点击 **[添加推广员]** 或列表右侧编辑按钮。
2. 关键字段：
   - **渠道类型**：从“推广渠道”中选择（推荐优先使用渠道配置，而不是手工填文字）
   - **姓名**：推广员姓名/昵称
   - **联系方式**：电话（可选）
3. 保存后可在“线下订单管理”的订单表单中选择推广员，用于归因与结算。

**C. 删除与辅助操作**
1. **删除**：点击删除按钮并确认（删除后相关订单历史归因可能受影响，请谨慎）。
2. **复制 ID**：列表 ID 列提供复制按钮，用于排查数据或对接外部系统时定位推广员。
3. **注意**：若无“查看所有推广人员”权限，默认只能看到自己创建的推广员。

#### 3.3.2 推广渠道（/commission）
**用途**：配置渠道与账号组的提成/结算规则，并把规则应用到统计报表。

**A. 账号组（Groups）**
1. **新增/编辑账号组**：配置名称、描述、是否按“已完成订单”作为结算口径。
2. **关联账号**：把系统账号分配到账号组，决定其结算规则归属。
3. **规则配置**：配置阶梯规则（按单量/GMV 等维度，以页面编辑器为准）。
4. **高客单提成**：可额外配置高客单相关提成比例，并在报表中单列展示。

**B. 渠道（Channels）**
1. **新增/编辑渠道**：配置渠道名称、启用/禁用、结算口径等。
2. **渠道规则**：为特定渠道配置规则（如启用后参与统计拆分）。
3. **删除渠道**：需二次确认，删除后相关数据统计口径会变化。

### 3.4 结算统计

#### 3.4.1 账号结算（/stats/accounts）
**用途**：按账号/账号组聚合订单与营收，并计算提成。

1. **周期切换**：支持累计/按月/自定义区间（按页面 Tab 选择）。
2. **筛选**：可按账号组筛选；非管理员通常只能查看自己名下订单汇总。
3. **拆分视图**：支持按渠道拆分到账号与推广员维度，显示单量、营收、提成、高客单提成等。
4. **导出明细**：可导出业绩明细/订单明细/高客单提成明细（文件名包含导出时间）。

#### 3.4.2 推广员结算（/stats/promoters）
**用途**：按推广员聚合订单与营收，并计算推广员提成。

1. **周期切换**：累计/按月/自定义区间。
2. **筛选**：可按渠道、账号组、账号筛选（视权限开放）。
3. **查看明细**：点击“查看明细”进入推广员的订单维度明细。
4. **导出报表**：导出推广员业绩明细（Excel）。

### 3.5 商品管理

#### 3.5.1 商品规格管理（/products）
**用途**：维护对外售卖的“商品型号/规格”，并通过 BOM 关联库存物品，支撑库存日历计算。

**A. 列表与检索**
1. **搜索**：按“商品型号名称”模糊搜索。
2. **分页**：支持切换每页行数与分页。
3. **查看详情**：可快速查看“映射关键词/规格概览/价格规则/BOM 构成”等信息。

**B. 添加规格（商品型号 + 多规格）**
1. 点击右上角 **[添加规格]**。
2. **商品型号名称**：对外展示的型号（如“大疆Pocket3”“iPhone 13”）。
3. **自动映射关键词（可选）**：用于线上订单自动关联商品；支持换行分隔（也可用逗号分隔）。当线上订单标题包含这些关键词时，系统会尝试自动关联到此商品。

**C. 规格配置（每个规格都要配）**
1. 点击 **[添加规格]** 增加一条规格。
2. 字段说明：
   - **规格名称**：如“标准版/高配版/单机/套装”
   - **规格ID**：系统用于内部识别；可留空让系统生成，也可手工输入（需要保持唯一）
   - **保险费(¥)**：该规格的保险费快照（用于订单金额计算）
3. **规格资产构成（BOM）**：
   - 点击 **[添加]**，为该规格选择库存物品类型并填写数量（xN）
   - BOM 决定库存日历占用与“规格库存”可用量计算
4. **阶梯价格表（天数-价格）**：
   - 点击 **[生成通用租期]** 自动生成常见天数档位
   - 点击 **[添加]** 手工增加价格规则；每条规则包含“天数/价格”
5. 点击底部 **[保存商品信息]** 完成保存。

**D. 同步旧规格（历史迁移）**
1. 点击 **[同步旧规格]**，用于把历史 variants 数据迁移到新的“规格 + BOM + 价格规则”结构。
2. 已经全部按新结构维护后，不需要频繁使用此功能。

**E. 删除**
1. 删除商品会影响订单录入与库存日历匹配，请谨慎操作；建议先确认无在途订单引用或已完成业务交接。

#### 3.5.2 库存数量管理（/inventory）
**用途**：维护仓库、库存物品定义，以及 SN/数量的入库出库；提供“库存与订单占用”的定位入口。

**A. 库存总览（合并视图）**
1. 展示每个物品类型的总库存与按仓库分布。
2. 展示序列化/非序列化物品统计与库存价值概览（按采购价估算）。

**B. 序列物品明细**
1. 支持按名称或 SN 搜索、按仓库/状态筛选。
2. **快速入库/批量入库**：一次录入多个 SN；系统自动创建序列物品记录。
3. **批量出库**：按 SN 列表执行出库；未找到 SN 会在结果提示。
4. **导入/导出 CSV**：用于批量初始化或备份序列物品清单（格式以页面提示为准）。
5. **SN 占用详情**：查看某个 SN 被哪些订单占用，并可一键跳转到对应订单列表定位。

**C. 非序列化物品（数量调整）**
1. 通过“库存数量调整”执行入库/出库（增加/减少）。
2. 可查看该物品类型关联的线下/线上订单占用明细。

#### 3.5.3 库存日历（/inventory-calendar）
**用途**：以日历/表格形式预测未来库存可用量，辅助接单与调度。

**A. 视图与筛选**
1. 顶部可在 **物品库存** 与 **规格库存** 两种视图间切换。
2. 在规格视图中可选择具体规格；若规格未配置 BOM，页面会提示并可一键跳转到规格配置。

**B. 日历含义**
1. **到**：当天归还到库数量
2. **发**：当天发货出库数量
3. **租**：当天在租订单数量
4. **库**：当天在库空闲数量（背景色：绿=充足 黄=紧张 红=缺货）

**C. 占用计算配置（仅可管理账号可见）**
1. 点击右上角设置按钮进入配置面板。
2. **默认发货缓冲/默认归还缓冲**：用于把实际占用区间向前/向后扩展。
3. **地区缓冲配置**：按省份配置专属缓冲天数；无匹配则使用默认值。
4. **过滤配置（赞晨商家过滤）**：填写需要过滤的商家名称（与订单中的“商家名称”一致），多个可用逗号/中文逗号/换行分隔；过滤后的订单不计入库存占用。
5. 点击 **[保存配置]** 生效。

### 3.6 系统管理

#### 3.6.1 账号权限管理（/users）
**用途**：维护系统账号与权限（RBAC）。

**A. 搜索与列表**
1. 顶部搜索框支持按 **姓名/用户名/权限名称** 检索。
2. 权限以标签方式展示（例如“线下订单管理/库存日历/系统设置”等）。
3. 支持分页与每页行数切换。

**B. 新增账号**
1. 点击 **[新增账号]**。
2. 填写姓名、用户名、密码。
3. 在“权限”区域按业务分组勾选（订单管理/推广管理/结算统计/商品管理/系统管理）。
4. 保存后刷新页面即可在左侧菜单看到该账号可访问的模块。

**C. 编辑账号**
1. 点击列表右侧编辑按钮。
2. 调整权限并保存；通常需要刷新页面后生效。

**D. 删除账号**
1. admin 账号不可删除。
2. 删除前请确认订单/推广员/数据导入导出等职责已交接，避免权限缺失导致无法操作。

#### 3.6.2 导出导入数据（/backup）
**用途**：对关键业务数据进行 JSON 导出/导入，便于迁移、恢复或环境初始化。

1. **可导出数据类型**（以页面勾选项为准）：
   - 库存配置（inventory.json）
   - 订单数据（orders.json）
   - 商品数据（products.json）
   - 推广员数据（promoters.json）
   - 用户账号（users.json）
   - 账号组配置（account-groups.json）
   - 推广渠道配置（channel-configs.json）
   - 提成规则（commission-rules.json）
   - 线上订单同步配置（online-orders-config.json）
2. **导出**：勾选要导出的数据类型，点击 **[导出选中数据]** 下载。
3. **导入**：点击 **[选择文件]** 上传 JSON；支持从本系统导出的文件恢复或合并数据（建议先导出一份当前数据作为备份）。
4. **操作记录**：页面下方展示导入/导出历史与执行结果。

#### 3.6.3 操作指南管理（/system/manual）
**用途**：在系统内维护可展示的操作指南章节（Markdown），用于后台自定义帮助内容。

1. 点击 **[新增章节]** 打开编辑面板。
2. 填写标题、排序、启用开关，并在“内容（Markdown）”中编写文档。
3. 右侧实时预览；保存后返回列表。
4. 支持编辑与删除章节。

#### 3.6.4 系统设置（/system/settings）
**用途**：生成与管理桌面工具连接所需的 API Token。

1. 在“API Token”区域点击 **生成 Token**（或重新生成）。
2. 点击复制按钮把 Token 复制到剪贴板。
3. 重新生成会使旧 Token 立即失效；桌面工具需要同步更新。

## 4. Sync Tool（桌面抓取工具）使用说明
Sync Tool 用于从各平台抓取线上订单并导入 ERP。其配置分“连接配置”和“平台配置”，并提供本地配置导入导出。

### 4.1 菜单：连接配置（Settings -> 连接配置）
1. **ERP 地址**：填写 Web 端访问地址（可省略 http://，末尾不含斜杠）。
2. **API Token**：在 Web 的“系统设置”生成并复制。
3. 点击 **保存**：会写入本地配置并尝试连接 ERP 拉取平台配置。
4. **导出本地配置**：一键导出为 JSON 文件（用于备份/迁移）。
5. **导入本地配置**：从 JSON 文件恢复本地配置；导入后建议重新点击“保存”以验证连接。

### 4.2 菜单：平台配置（Settings -> 平台配置）
1. 点击 **拉取 ERP 配置** 获取平台列表。
2. 在某个平台下可填写本地覆盖（如登录 URL/账号密码/选择器等）；留空则沿用 ERP 端配置。
3. 点击 **保存本地配置**：仅保存本地覆盖，不会修改 ERP 端的站点配置。

### 4.3 页面：抓取（Sync）
1. 顶部 **刷新配置**：重新拉取 ERP 端平台配置。
2. 顶部 **重启调度器**：定时抓取不触发时使用（重启本地定时任务）。
3. 平台列表：
   - **抓取/停止**：手动触发或停止单个平台抓取
   - **定时**：点击“编辑”按钮编辑该平台定时（HH:mm，逗号分隔）；显示“本地”代表使用本地覆盖时间
   - **显示/隐藏浏览器**：可按平台控制是否弹出浏览器窗口（便于排查验证码/风控）
   - **只看此平台日志**：快速过滤日志，定位问题
4. 日志区：展示抓取过程、导入结果与错误信息（最多保留最近一段日志）。

- stats_accounts：账号结算
- stats_promoters：推广员结算
- products：商品规格管理
- inventory_calendar：库存日历
- inventory_manage：库存数量管理
- users：账号权限管理
- backup：导出导入数据
- manual：操作指南管理
- system_settings：系统设置
